# 📱 Calculator App

간단한 사칙연산 기능을 제공하는 계산기 애플리케이션입니다.


## 📌 주요 기능
- 덧셈, 뺄셈, 곱셈, 나눗셈
- 실시간 연산 결과 표시
- 클리어(초기화) 기능

## 📦 기술 스택
- Python 3.x
- Tkinter (GUI 인터페이스용)
- 기타 사용 라이브러리 (필요시 추가)
